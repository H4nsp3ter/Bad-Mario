# 🍄 Bad Mario: Metal Mayhem 🎸

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Build: Pure JS](https://img.shields.io/badge/Engine-Vanilla%20JS-orange.svg)]()
[![Vibe: Brutal](https://img.shields.io/badge/Vibe-Brutal%20Metal-red.svg)]()

Forget rescuing princesses. In **Bad Mario**, the Mushroom Kingdom has fallen—overrun by the undead and corrupt military forces. Mario has had enough. With a cigar in his mouth and an arsenal that would make Slayer proud, it's time to clean up the streets.

---

## 🔥 Features

* **🎸 Heavy Metal Soundtrack:** A dynamic BGM system featuring real metal tracks. The music shifts into high-gear Thrash mode the moment a boss enters the arena.
* **🔫 Arsenal Inventory:** Collect and stack weapons like Miniguns, Flamethrowers, and Rocket Launchers. Manage your gear and swap weapons tactically using `Q`.
* **🧟 Brutal Gore System:** Realistic (yet pixelated) particle effects, blood splatters on the "camera lens," and flying shell casings.
* **👹 Epic Bosses:** Face off against massive, mutated monstrosities at the end of every level, each with unique attack patterns.
* **🍻 Badass Collectibles:** No more shiny gold coins. Mario collects Beer and hard Liquor to fuel his rampage and boost the high score.
* **🕹️ 16-Bit Arcade Vibe:** CRT scanlines, massive screen shake, and chunky retro graphics.

## 🛠️ Technical Overview

The game is built entirely in **Vanilla JavaScript**, leveraging the power of the HTML5 Canvas API and Web Audio API. No external frameworks, no engines—just pure code.

* **Procedural Level Generation:** Every level is generated on the fly, including seamless Boss Arenas and dynamic difficulty scaling.
* **Hybrid Audio Engine:** A blend of high-quality MP3 samples for the music and procedurally synthesized sounds for gunshots and screams.
* **Live Sprite Generator:** All graphics are dynamically drawn via code at runtime—ensuring a tiny footprint with a maximum retro look.

## 🚀 Installation & Setup

Since Bad Mario has zero external dependencies, getting it running is simple:

1.  Clone the repository or download the ZIP.
2.  Ensure your metal tracks (`level_metal_1.mp3`, etc.) are in the root directory.
3.  Open `index.html` in a modern browser (Chrome/Edge recommended for optimal Web Audio support).
4.  **IMPORTANT:** Click anywhere on the screen once to allow the browser to play audio!
